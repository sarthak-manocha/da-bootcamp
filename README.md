Data Analyst Bootcamp Projects

This repository contains hands-on projects and learning materials completed as part of the Data Analyst Bootcamp by Alex The Analyst. It demonstrates practical, job-ready skills across key data analytics tools including Excel, SQL, Python, Tableau, and Power BI.

Overview

This repository highlights end-to-end data analytics capabilities, including:

Data cleaning and preprocessing
Exploratory data analysis (EDA)
Data visualization and dashboard development
SQL-based data querying and transformation
Business insights generation and reporting

All work is based on real-world datasets and scenarios relevant to data analyst roles.
